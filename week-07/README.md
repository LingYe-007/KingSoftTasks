作业在实现的过程中,还是遇到了一些问题,大多已解决,还有部分有些不懂：



\1. React用import 导入样式,实际上,组件之间的样式还是会互相作用,而不像vue中用style scoped 可以控制css的作用域,就感觉挺奇怪的,当组件多了复杂了,没有控制css作用域的东西不会很难受吗？
